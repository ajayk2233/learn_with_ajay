Download & Install Python

Follow the instruction to download latest version of python.

1. Click here to go to the Official website & follow the steps

2. click on to download it for Windows.
You can find the link to download it for Mac/Linux.

3. Find the file in your standard download folder & open it.

The following window will appear.

4. Click Install Now.

A new Python 3.8.3 (64-bit) Setup pop-up window will appear with a Setup Progress message and a progress bar.

5. Now click on Close.

You have successfully installed Python in your computer.

6. Lets check whether installation went correctly.

Goto CMD (press windows + r combination keys) & type python --version

if you see the version shown in above image then Congratulations... You have done with Installation.
